08 // the approach

Full Schema Copy

Direct SQL cross-schema replication. No per-model logic, no missed data.

Direct SQL

INSERT INTO dest SELECT FROM source - copies every table in one pass. No ORM overhead, no model-by-model iteration.

Column-safe

Queries column names from information_schema. Handles schemas with different column ordering gracefully.

Drift-safe

Intersects source and destination tables. Orphaned tables from deleted migrations are skipped automatically.

How it works: 1. Get tables intersect(source_tables, dest_tables) 2. For each table: a. Get columns intersect(source_cols, dest_cols) b. Truncate TRUNCATE dest.table CASCADE c. Copy INSERT INTO dest.table(cols) SELECT cols FROM source.table 3. Done complete fidelity, all relationships preserved
