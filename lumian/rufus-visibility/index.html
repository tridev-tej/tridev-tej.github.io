Anti-Detection

Five layers between us and a captcha.

L1 · Pacing Gaussian sleep 90-240s between queries, μ=165s, σ=45s. No fixed intervals.
L2 · Cap 8 / hour Hard hourly limit per account, persisted across runs.
L3 · CSRF 3h auto-refresh anti-csrftoken-a2z extracted on every cookie-refresh cron. Rufus CSRF TTL ~6h — comfortable margin.
L4 · Jitter Pre-query delay Random 0.5-3s pause before each request. Breaks robotic timing patterns at the runner.
L5 · Block detect Hard stop Captcha keywords → exit immediately. No retries through blocks.
