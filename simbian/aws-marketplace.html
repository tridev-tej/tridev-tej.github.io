02 // the integration

Subscribe to Live in Under 5 Minutes

AWS Marketplace subscription handling

Webhook receives subscription event. Validates with AWS using seller account credentials.

Self-service registration page

Clean registration form with inline CSS (no tenant stylesheets available yet). Loading spinner on submit.

Auto-provisioning

Creates tenant schema, seeds default data, sets up admin user. All in one flow.

Seller credential isolation

Uses Simbian's seller AWS credentials for Marketplace API calls. Separate from customer's AWS credentials.

03 // challenges solved

Multi-Tenant Registration Is Tricky

Public schema context

Registration pages must run in public schema (no tenant yet). Special middleware handling.

Inline CSS only

Tenant stylesheets don't exist during registration. All styling must be inline.

Credential management

Seller account credentials stored as env vars. Not mixed with customer credentials.

Backport to release

Feature needed on both main and release branches. Cherry-pick strategy for hotfixes.
